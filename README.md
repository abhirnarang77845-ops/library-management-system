# Library Management System

A mini library management system built in Python using dictionaries.

## Features
- Add Books
- Show All Books
- Issue Books (with student name and date)
- Return Books (with fine calculation)

## Fine System
- Week 1 → Rs. 10/day
- Week 2 → Rs. 20/day
- Week 3 → Rs. 60/day

## How to Run
python main.py
